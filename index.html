Cifrado César

Ingresa un texto y una llave numérica. Presiona Cifrar o Descifrar. Se conservan mayúsculas/minúsculas y símbolos.

Python en el navegador · GitHub Pages

Operación

key = key % 26
0–25
Tip: si pones 26, queda igual. Si pones -3, es como descifrar con 3.

Guía rápida

Ejemplo "ABC", key=3 → "DEF"
Wrap "xyz", key=2 → "zab"
Descifrar encrypt(text, -key)

Esta interfaz corre en el navegador con PyScript, así que se despliega como sitio estático en GitHub Pages.
